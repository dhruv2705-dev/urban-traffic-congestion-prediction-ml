# AI-Driven Urban Traffic Congestion Prediction System

## 📌 Overview
This project is an AI-based urban traffic management system designed to predict congestion levels and optimize city mobility. It uses machine learning models along with an interactive Streamlit dashboard to provide real-time insights and route optimization.

## 🚀 Features
- Traffic congestion prediction using Machine Learning
- Interactive dashboards using Streamlit and Plotly
- Role-based system (User & Government Authority)
- Route optimization and map visualization
- Traffic chatbot for intelligent queries
- Congestion heatmaps and analytics
- Infrastructure planning support

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Plotly
- Folium (Maps)
- Joblib

## ⚙️ System Modules
- **Predictive Engine**: Predict congestion (Low, Medium, High)
- **User Advisory System**: Route planning and travel suggestions
- **Government Dashboard**: Traffic analytics and planning tools
- **Traffic Chatbot**: AI-based query assistant
- **Heatmaps & Visualization**: Data-driven insights

## 📊 Machine Learning
- Model: Random Forest Classifier
- Features: Traffic volume, speed, weather, incidents, road usage
- Output: Congestion category prediction
